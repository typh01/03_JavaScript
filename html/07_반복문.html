for문

    - 프로그램의 흐름을 제어하는 제어문 중 하나.

    - 특정 코드를 반복 수행할 때 사용하는 구문

    - 끝이 정해져 있는 반복에서 주로 사용
    ex) 운동장 10바퀴 돌기
        -> 반복할 횟수가 정해져 있음 
    

    [작성법]

    for([1]초기식 ; [2]조건식 ; [4]증감식 ){

      // [3] 조건식이 true일 경우 수행할 코드;

    }

    * 처음 for문이 수행 될 때만 [1] 초기식을 수행
      -> 두 번째 반복부터는 [2],[3],[4] 만 반복

    초기식 : for문을 제어하는 용도의 변수 선언

    조건식 : for문의 반복 여부를 지정하는 식
            
            - true일 때 반복,
              false가 되는 순간 바로 반복(for) 종료

            - 보통 초기식에 사용된 변수를 이용해서
              조건식을 작성함

    증감식 : 초기식에 사용된 변수 값을 변환 시키는 식

            - 변환된 초기식에 의해서 조건식의 결과가 달라지게 함함

콘솔에 12345 출력하기

콘솔에 1부터 10까지 1씩 증가하며 출력하기

콘솔에 5부터 13까지 1씩 증가하며 출력하기

1부터 7까지 1씩 증가한 문자열을 한 줄로 출력하기

4부터 11까지 1씩 증가한 문자열을 한 줄로 출력하기

1부터 10까지 1씩 증가한 숫자의 합 구하기

입력 받은 값 사이 정수 값들의 합 구하기

입력 1 :
입력 2 :

1부터 10까지 2씩 증가한 숫자 출력 + 합계계 구하기

3부터 30까지 3의 배수만 출력하기

입력 받은 범위 내 지정된 수의 배수를 출력, 합계 구하기

시작 :
종료 :
배수 :

구구단 2단 출력하기

    2 x 1 = 2
    2 x 2 = 4
    2 x 3 = 6
    ....
    2 x 9 = 18

입력 받은 단의 구구단 출력하기

    - 단이 입력되지 않으면
      -> alert("단을 입력 해주세요"); 출력 후 함수 종료
    
    - 입력된 단이 2 ~ 9단이 아니면
      -> alert("2 ~ 9단 사이만 입력 해주세요"); 출력 후 함수 종료
단 입력 :

10부터 1까지 1씩 감소하며 출력하기

20부터 3까지 3씩 감소하며 출력하기

구구단 3단 거꾸로 출력하기

    3 x 9 = 27
    3 x 8 = 24
    3 x 7 = 21 
    ....
    3 x 1 = 3

5의 배수마다 @, 10의 배수마다 줄바꿈하면서 출력하기

    [결과 화면]
    1 2 3 4 @ 6 7 8 9 @
    11 12 13 14 @ 16 17 18 19 @
    21 22 23 24 @ 26 27 28 29 @
