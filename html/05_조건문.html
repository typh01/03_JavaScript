조건문

    - 프로그래밍 코드가 해석되는 순서를 제어하는 "제어문" 중 한 종류

    - 작성된 [조건식]에 따라서 수행되는 코드를 변경하는 구문

    * 조건식 : 결과가 true 또는 false인 식
    ex) 논리 연산자(&&, ||, !)
        비교 연산자(>, >=, !=, ...)
        함수 호출 결과로 boolean 반환
        boolean 단독 작성

if(만약에...) 문

    
      if(조건식){
        조건식이 true인 경우 수행할 코드;
      }

입력 받은 수가 양수(0보다 큰 수)인지 확인

if(만약에) - else(아니면) 문

    
      if(조건식){
        조건식이 true인 경우 수행할 코드;
      }

      else {
        // else는 (조건식) 작성X
        
        조건식이 false인 경우 수행할 코드;
      }

1 ~ 100 사이의 난수를 발생 시켜 홀/짝 판별

if(만약에) - else if(앞 조건이 아니라면 다른 조건) - else 

    
    if(조건식){
      조건식이 true인 경우 수행할 코드;
    }

    else if(조건식){
      앞선 if 또는 else if의 조건식이 false 이면서,
      현재 else if는 true인 경우 수행할 코드;
    }

    ...

    else{
      앞선 모든 if, else if가 false 인 경우 수행할 코드;
    }
